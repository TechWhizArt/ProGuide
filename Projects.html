Rules for project submission.

To submit your project for display on our website, follow these simple rules:

What to Submit

Project Details:

  • Project Title: Name of your project.
  • Languages Used: List the tools or programming languages used.
  • Website Link: Your project must be live on GitHub Pages.
  • Code Repository: Provide the public GitHub repository link.

Personal Details:

  • Full Name
  • Semester
  • Date of Submission

Submission Rules

  • Quality: The website must work correctly and have clean, well-documented code.
  • Original Work: Submit only your own projects. Plagiarized work will be rejected.
  • Deadline: Submit your projects befire 11:59pm on Saturdays as new projects will be added to the website every Sunday altogether.
  • Contact Details: Ensure your GitHub profile has valid contact information.

How to Submit

Send all required details and links to us via email. Approved projects will be showcased on our website.
